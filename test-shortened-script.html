üîß How It Works

  1. Widget loads with only the botId
  2. Widget.js fetches full config from API: GET /api/bot-config/{botId}
  3. Widget initializes with the fetched configuration
  4. Config is cached for 5 minutes for better performance

üåê API Endpoints

Bot Configuration API

  • GET /api/bot-config/{botId} - Get bot config (public)
  • POST /api/bot-config/{botId} - Save bot config
  • PUT /api/bot-config/{botId} - Update bot config
  • DELETE /api/bot-config/{botId} - Delete bot config
  • GET /api/bot-configs - List all configs
